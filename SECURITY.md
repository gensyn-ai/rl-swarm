# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in RL Swarm:

- Do not open a public GitHub issue.
- Use the **“Report a vulnerability”** option in the Security tab of this repository, or
- Contact the maintainers via the official channels listed at https://docs.gensyn.ai.

Please include:

- A description of the issue  
- Steps to reproduce  
- Relevant logs and environment details (OS, hardware, Python version)

We will review the report and work on a fix as quickly as possible.
